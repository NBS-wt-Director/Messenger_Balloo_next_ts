## 🚀 Пошаговый План Релиза

### Шаг 1: Безопасность (15 минут)

```bash
# 1. Сгенерировать новый JWT_SECRET
cd messenger
npm run dev
# Открыть консоль браузера и выполнить:
crypto.randomUUID() + crypto.randomUUID()

# 2. Отредактировать config.json
# JWT_SECRET: "сгенерированное-значение-из-консоли"
# defaultAdminPassword: "СложныйПароль!2024#XyZ"

# 3. Добавить config.json в .gitignore
echo "config.json" >> .gitignore
```

### Шаг 2: Проверка База Данных (10 минут)

```bash
# Остановить dev сервер (Ctrl+C)
# Удалить старую БД IndexedDB:
# 1. Откройте http://localhost:3004
# 2. F12 → Application → IndexedDB → balloo → Delete

# Перезапустить
npm run dev
```

### Шаг 3: Финальная Сборка (5 минут)

```bash
# Остановить dev сервер
# Очистить кэш
rm -rf .next

# Пересобрать
npm run build

# Проверить
npm run lint
npx tsc --noEmit
```

### Шаг 4: Деплой (15-30 минут)

**Вариант A: Vercel (рекомендуется)**
```bash
npm i -g vercel
vercel --prod
```

**Вариант B: Railway**
1. Подключить GitHub
2. Добавить переменные окружения
3. Деплой автоматически

**Вариант C: VPS**
```bash
# На сервере
cd /var/www/balloo
npm install --production
npm run build
pm2 start npm --name balloo -- start
```

### Шаг 5: Пост-Деплой (10 минут)

1. ✅ Создать админа:
```bash
npm run create-admin
```

2. ✅ Проверить все страницы:
- `/` - Главная
- `/login` - Вход
- `/register` - Регистрация
- `/chats` - Чаты
- `/admin` - Админка
- `/support` - Поддержка
- `/features` - Функции

3. ✅ Проверить изображения:
```bash
curl -I https://yourdomain.com/logo.png
curl -I https://yourdomain.com/mascot.png
```
