## ⚠️ Что нужно сделать перед деплоем

### Обязательно

1. **Сгенерировать секреты**
   ```bash
   # JWT Secret
   openssl rand -base64 32
   
   # Encryption Key
   openssl rand -hex 32
   ```

2. **Настроить переменные окружения**
   - Скопировать `.env.example` → `.env.production`
   - Заполнить все обязательные переменные
   - Проверить `DATABASE_URL`

3. **Yandex OAuth**
   - Создать приложение на [oauth.yandex.ru](https://oauth.yandex.ru)
   - Настроить redirect URI
   - Получить Client ID и Client Secret

4. **База данных**
   - Для production: PostgreSQL вместо SQLite
   - Создать БД и пользователя
   - Запустить миграции: `npm run db:setup`

5. **Домен и SSL**
   - Настроить домен
   - Получить SSL сертификат (Let's Encrypt)
   - Проверить HTTPS

### Рекомендовано

1. **Мониторинг**
   - Подключить Sentry для ошибок
   - Настроить Uptime мониторинг
   - Настроить алерты

2. **Бэкапы**
   - Настроить автоматические бэкапы БД
   - Проверить восстановление из бэкапа

3. **Push-уведомления**
   - Сгенерировать VAPID ключи
   - Настроить в `.env`

4. **Email**
   - Настроить SMTP сервер
   - Проверить отправку email

## 🚀 Быстрый деплой

### Вариант 1: Vercel (Рекомендуется)

```bash
# 1. Установите Vercel CLI
npm i -g vercel

# 2. Войдите
vercel login

# 3. Деплой
vercel --prod
```

**Время деплоя:** 3-5 минут  
**Сложность:** ⭐☆☆ (Легко)

### Вариант 2: Railway

```bash
# 1. Установите Railway CLI
npm i -g @railway/cli

# 2. Войдите
railway login

# 3. Инициализируйте проект
railway init

# 4. Добавьте переменные окружения
railway variables set NODE_ENV=production
railway variables set JWT_SECRET=your-secret

# 5. Деплой
railway up
```

**Время деплоя:** 5-7 минут  
**Сложность:** ⭐☆☆ (Легко)

### Вариант 3: Собственный сервер

```bash
# 1. Клонируйте репозиторий
git clone <repo> && cd messenger

# 2. Установите зависимости
npm install --production

# 3. Настройте окружение
cp .env.example .env.production.local
nano .env.production.local

# 4. Инициализируйте БД
npm run db:setup

# 5. Запустите через PM2
pm2 start npm --name balloo -- start
pm2 save
pm2 startup
```

**Время деплоя:** 20-30 минут  
**Сложность:** ⭐⭐⭐ (Средне)
