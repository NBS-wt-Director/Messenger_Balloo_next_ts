## 📋 ЧТО СДЕЛАНО

### 1. ✅ Конфигурация через JSON
- **Файл:** `config.json`
- **Заменяет:** `.env` переменные
- **Преимущества:** 
  - Все настройки в одном месте
  - Легко копировать между окружениями
  - Версионируется в git

### 2. ✅ Тестовые пользователи
- **4 пользователя** созданы
- **1 супер-админ** + 3 обычных
- **Пароли запомнены** (см. ниже)

### 3. ✅ Тестовые чаты
- **3 личных чата** между пользователями
- **1 групповой чат** (все участники)
- **5+ тестовых сообщений**

## 🧪 БЫСТРЫЙ ТЕСТ

### 1. Войти как Админ
```
1. Открыть http://localhost:3000/login
2. Email: admin@balloo.ru
3. Пароль: Admin123!
4. Нажать "Войти"
```

### 2. Проверить Чаты
```
1. Должны быть 3 системных чата:
   - Избранное
   - Техподдержка
   - Balloo - новости
2. Должны быть пользовательские чаты:
   - chat_alex_maria
   - chat_alex_dima
   - chat_maria_dima
   - chat_all_group
```

### 3. Открыть чат
```
1. Кликнуть на любой чат
2. Отправить сообщение
3. Проверить статус "Отправлено"
```

### 4. Проверить Админку
```
1. Открыть http://localhost:3000/admin
2. Должна открыться админ-панель
3. Проверить вкладку "Жалобы"
```

### 5. Проверить Настройки
```
1. Открыть http://localhost:3000/settings
2. Сменить тему (Тёмная/Светлая/Россия)
3. Проверить PWA статус
4. Включить уведомления
```

## 🐛 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Ошибка: "Module not found"
```bash
npm install
```

### Ошибка: "Database error"
```bash
# Очистить IndexedDB в браузере
# Application → IndexedDB → Delete
```

### Ошибка: "Unauthorized"
```bash
# Проверить config.json
# Убедиться, что JWT_SECRET настроен
```

### Ошибка: "Port 3000 already in use"
```bash
# Изменить порт в package.json
# "dev": "next dev -p 3001"
```
