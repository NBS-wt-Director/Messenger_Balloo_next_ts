### 2. 🎫 Страница "Мои Приглашения"

#### Расположение:
- **URL:** `/invitations`
- **Кнопка в навигации:** рядом с "Контакты"

#### Функционал:
- ✅ **Список приглашений** с карточками
- ✅ **Статусы приглашений:**
  - 🟢 Активно (зелёный)
  - 🟡 Использовано (жёлтый)
  - ⚫ Истекло/Деактивировано (серый)
- ✅ **Предпросмотр ссылки:**
  - URL в поле input
  - Кнопка "Скопировать" с анимацией
  - Кнопка "Открыть в новой вкладке"
- ✅ **Статистика:**
  - Использовано / Лимит
  - Дата истечения
  - Название чата
- ✅ **Фильтры:**
  - Все
  - Активные
  - Истекшие
- ✅ **Создание приглашения:**
  - Выбор чата
  - Настройка лимита
  - Срок действия
  - Одноразовое/многоразовое
- ✅ **Деактивация** приглашений

#### API Endpoints:
- `GET /api/invitations/my?userId=USER_ID` - список приглашений
- `POST /api/invitations/my` - создание приглашения
- `DELETE /api/invitations/my?id=ID` - деактивация
