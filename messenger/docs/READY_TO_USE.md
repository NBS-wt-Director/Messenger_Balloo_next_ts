## 🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

### 1. Ошибка: "Module not found"
**Решение:**
```bash
npm install
```

### 2. Ошибка: "Database error"
**Решение:**
```
1. Открыть DevTools (F12)
2. Application → IndexedDB
3. Delete "balloo"
4. Обновить страницу
```

### 3. Ошибка: "Unauthorized"
**Решение:**
```
Проверить config.json:
auth.jwtSecret должен быть настроен
```

### 4. Ошибка: "Port 3000 already in use"
**Решение:**
```bash
# Изменить порт в package.json
"dev": "next dev -p 3001"
```
