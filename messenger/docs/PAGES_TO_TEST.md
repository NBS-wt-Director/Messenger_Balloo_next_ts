## ✅ ЧЕК-ЛИСТ ДЛЯ ПРОВЕРКИ

### Базовый функционал

- [ ] **Регистрация** - Создать нового пользователя
- [ ] **Логин** - Войти под существующим
- [ ] **Отображение чатов** - 3 системных чата + пользовательские
- [ ] **Переключение вкладок** - Чаты/Контакты/Приглашения
- [ ] **Поиск** - Поиск по чатам
- [ ] **Избранное** - Добавление в избранное

### Чаты

- [ ] **Открыть чат** - Переход в чат
- [ ] **Отправить сообщение** - Текстовое сообщение
- [ ] **Статусы сообщений** - Отправлено/Доставлено/Прочитано
- [ ] **Закрепить чат** - Pin чата (макс 15)
- [ ] **Очистить историю** - Удаление сообщений
- [ ] **Заблокировать** - Блокировка пользователя
- [ ] **Пожаловаться** - Отправка жалобы

### Приглашения

- [ ] **Создать приглашение** - Генерация ссылки
- [ ] **Копировать ссылку** - Копирование в буфер
- [ ] **Принять приглашение** - Переход по ссылке
- [ ] **Деактивировать** - Отключение приглашения

### Настройки

- [ ] **Смена темы** - Тёмная/Светлая/Россия
- [ ] **Смена языка** - RU/HI/ZH/TT
- [ ] **Уведомления** - Вкл/Выкл + разрешение
- [ ] **PWA** - Проверка установки
- [ ] **Яндекс.Диск** - Подключение (если настроено)

### Админ-панель (для admin@balloo.ru)

- [ ] **Доступ** - Вход только для админа
- [ ] **Пользователи** - Список всех пользователей
- [ ] **Жалобы** - Просмотр и обработка
- [ ] **Статистика** - Просмотр статистики

## 🧪 ТЕСТОВЫЕ СЦЕНАРИИ

### Сценарий 1: Личная переписка
1. Войти как **user1@balloo.ru**
2. Открыть чат с **Марией**
3. Отправить сообщение
4. Проверить статус "Отправлено"
5. Выйти, войти как **user2@balloo.ru**
6. Проверить получение сообщения
7. Ответить
8. Проверить статус "Прочитано" у первого пользователя

### Сценарий 2: Групповой чат
1. Войти как **user1@balloo.ru**
2. Открыть "Общая группа"
3. Отправить сообщение
4. Проверить реакции (👍)
5. Закрепить чат

### Сценарий 3: Приглашение
1. Войти как **user1@balloo.ru**
2. Перейти в "Приглашения"
3. Создать новое приглашение
4. Скопировать ссылку
5. Открыть в инкогнито
6. Принять приглашение

### Сценарий 4: Жалоба
1. Войти как **user2@balloo.ru**
2. Открыть чат
3. Нажать "Пожаловаться"
4. Выбрать причину
5. Отправить
6. Войти как **admin@balloo.ru**
7. Открыть `/admin/reports`
8. Обработать жалобу

### Сценарий 5: PWA и Уведомления
1. Открыть `/settings`
2. Проверить PWA статус
3. Включить уведомления
4. Разрешить в браузере
5. Отправить тестовое уведомление
