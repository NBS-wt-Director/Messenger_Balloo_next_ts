## 🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### 1. Ошибка: "Module not found"
**Решение:**
```bash
npm install
```

### 2. Ошибка: "Database error"
**Решение:**
```
1. F12 → Application
2. IndexedDB → Delete "balloo"
3. Обновить страницу
```

### 3. Ошибка: "Unauthorized"
**Решение:**
```
Проверить config.json:
auth.jwtSecret должен быть настроен
```

### 4. Ошибка: "Port 3000 already in use"
**Решение:**
```bash
npm run dev -- -p 3001
```
