## 📁 Куда Добавить Картинки

### 1. Логотип (`logo.png`)

**Путь:** `messenger/public/logo.png`

**Требования:**
- Размер: **200x200px** или **400x400px**
- Формат: **PNG с прозрачным фоном**
- Вес: < 100KB

**Где используется:**
- Header (верхняя панель)
- Footer (подвал)
- AuthPage (страница входа)

---

### 2. Маскот (`mascot.png`)

**Путь:** `messenger/public/mascot.png`

**Требования:**
- Размер: **100x100px** или **200x200px**
- Формат: **PNG с прозрачным фоном**
- Вес: < 50KB

**Где используется:**
- BurgerMenu (кнопка меню в хедере)

---

## 🔲 Заглушка (По Умолчанию)

Если файлы **не добавлены**, компоненты автоматически показывают **красно-бело-синий квадрат**:

### Логотип без картинки:
```
┌─────────────┐
│  🎈 Balloo  │
└─────────────┘
```
- Градиентный квадрат (красный-белый-синий)
- Иконка сообщения
- Текст "Balloo"

### Бургер меню без маскота:
```
┌────┐
│ ≡  │ ← иконка меню поверх градиента
└────┘
```
- Градиентный квадрат (красный-белый-синий)
- Иконка меню (3 полоски)

## 🎨 Адаптивность

Компоненты автоматически адаптируются:

| Экран | Логотип | Бургер меню |
|-------|---------|-------------|
| Desktop (>768px) | 40px + текст | 40px |
| Mobile (<768px) | 32px без текста | 32px |

---

## 📝 Пример Добавления

### 1. Положите файлы в папку `public/`:
```
messenger/
├── public/
│   ├── logo.png      ← ваш логотип
│   └── mascot.png    ← ваш маскот
```

### 2. Компоненты автоматически подхватят файлы:
- Логотип: `<Logo src="/logo.png" />`
- Бургер: `<BurgerMenu mascotSrc="/mascot.png" />`

### 3. Если файлы не найдены - покажется заглушка

---

## 🛠 Технические Детали

### Компонент Logo

```tsx
<Logo 
  src="/logo.png"      // путь к картинке (опционально)
  alt="Balloo"         // alt текст
  size="md"            // sm | md | lg
  showText={true}      // показывать текст
  className=""         // доп. CSS классы
/>
```

### Компонент BurgerMenu

```tsx
<BurgerMenu 
  mascotSrc="/mascot.png"  // путь к маскоту (опционально)
  mascotAlt="Mascot"       // alt текст
  size="md"                // sm | md | lg
  isOpen={false}           // открыто ли меню
  onToggle={() => {}}      // обработчик клика
  className=""             // доп. CSS классы
/>
```

### Логотип:
- Простой и узнаваемый
- Хорошо смотрится в малом размере (32px)

- Анимированная версия (опционально)

Если видите красно-бело-синие квадраты - файлы не найдены.
