# 📁 Папка для изображений

## 🎨 Логотип и Маскот

### Файлы для добавления:

1. **`logo.png`** - Логотип мессенджера
   - Рекомендуемый размер: **200x200px** или **400x400px**
   - Формат: **PNG с прозрачным фоном**
   - Используется в: Header, Footer, AuthPage

2. **`mascot.png`** - Маскот для бургер меню
   - Рекомендуемый размер: **100x100px** или **200x200px**
   - Формат: **PNG с прозрачным фоном**
   - Используется в: BurgerMenu (кнопка меню)

### 🔲 Заглушка (по умолчанию)

Если файлы не добавлены, компоненты автоматически показывают **красно-бело-синий квадрат**:
- Логотип: градиентный квадрат + иконка сообщения + текст "Balloo"
- Бургер меню: градиентный квадрат + иконка меню

### 📐 Адаптивность

Компоненты автоматически адаптируются под разные размеры экранов:
- **Desktop**: полный размер с текстом
- **Mobile**: уменьшенный размер без текста

### 🎯 Где используются:

| Компонент | Файл | Изображение |
|-----------|------|-------------|
| Header | `src/components/Header.tsx` | logo.png |
| BurgerMenu | `src/components/ui/BurgerMenu.tsx` | mascot.png |
| Footer | `src/components/Footer.tsx` | logo.png (опционально) |
| AuthPage | `src/components/pages/AuthPage.tsx` | logo.png (опционально) |

## 📷 Другие изображения

### Аватарки пользователей
- Сохраняются через API: `/api/profile/avatar`
- Путь: определяется сервером

### Вложения
- Загружаются на Яндекс.Диск через API
- Путь: определяется Яндекс.Диск API

### Иконки PWA
- Папка: `/public/icons/`
- Файлы: `icon-192.png`, `icon-512.png`

## 🛠 Технические требования

- **Формат**: PNG (рекомендуется), SVG, WebP
- **Размер файла**: < 100KB для логотипа, < 50KB для маскота
- **Цвета**: соответствуют теме приложения (см. `globals.css`)
- **Прозрачность**: рекомендуется для лучшего вида
