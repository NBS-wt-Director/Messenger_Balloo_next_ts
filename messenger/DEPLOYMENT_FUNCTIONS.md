## Созданные API Endpoints (12 новых)

### Аутентификация
1. `POST /api/auth/email/verify` - Отправить код подтверждения email
2. `PUT /api/auth/email/verify` - Подтвердить email
3. `POST /api/auth/password/recovery` - Запрос восстановления пароля
4. `POST /api/auth/password/reset` - Сброс пароля по токену
5. `GET /api/auth/password/reset/validate` - Проверка токена

### Профиль
6. `POST /api/profile/avatar/upload` - Загрузка аватарки с обновлением профиля

### Группы
7. `POST /api/chats/group/create` - Создание группы с настройками
8. `GET /api/chats/group/members` - Получить участников группы
9. `POST /api/chats/group/members` - Добавить участников
10. `DELETE /api/chats/group/members` - Удалить участника/выйти
11. `PUT /api/chats/group/role/update` - Назначить роль
12. `GET /api/chats/group/role/list` - Список ролей и прав

### Вложения
13. `POST /api/yandex-disk/upload/video` - Загрузка видео
14. `GET /api/yandex-disk/video/:path` - Просмотр видео
15. `POST /api/yandex-disk/upload/document` - Загрузка документа
16. `GET /api/yandex-disk/download` - Скачивание документа
17. `POST /api/attachments/preview` - Предпросмотр вложений
18. `GET /api/attachments/preview/video/:id` - Метаданные видео

### Пересылка
19. `POST /api/messages/forward` - Пересылка сообщения
20. `GET /api/messages/forward/:id` - Оригинальное сообщение

### Яндекс.Диск
21. `GET /api/yandex-disk/auth` - URL авторизации
22. `POST /api/yandex-disk/link` - Связать аккаунт
23. `DELETE /api/yandex-disk/unlink` - Отвязать аккаунт
24. `GET /api/yandex-disk/status` - Статус подключения

## Созданные компоненты (3 новых)

1. **GroupMembersManager.tsx** - Управление участниками группы
   - Просмотр списка участников
   - Добавление/удаление участников
   - Назначение ролей
   - Выход из группы

2. **ChangePasswordModal.tsx** - Смена пароля (уже существовал, проверен)
   - Проверка текущего пароля
   - Валидация нового пароля
   - Подтверждение пароля

3. **Страницы восстановления пароля**
   - `/forgot-password` - Запрос восстановления
   - `/reset-password` - Сброс пароля

## Обновлённые роли в группах

| Роль | Права |
|------|-------|
| **creator** | Полный доступ: управление участниками, ролями, группой, удаление |
| **moderator** | Управление участниками, удаление сообщений |
| **author** | Отправка и чтение сообщений |
| **reader** | Только чтение сообщений |
