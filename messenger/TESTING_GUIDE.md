# Тестирование функционала Balloo Messenger

## 1. ТЕСТ: Создание нового чата и автоматическое добавление в контакты

### Шаги:
1. Открой `https://alpha.balloo.su/chats`
2. Нажми кнопку **"+"** (создать чат)
3. Выбери пользователя из списка
4. Создай чат

### Ожидаемый результат:
- ✅ Чат создан
- ✅ Получатель автоматически добавлен в мои контакты
- ✅ Я автоматически добавлен в контакты получателя (зеркально)
- ✅ Создана семейная связь (relationType: 'friend')

---

## 2. ТЕСТ: Отправка и получение сообщений

### Шаги:
1. Открой любой чат
2. Введи текст сообщения
3. Нажми Enter или кнопку отправки
4. Открой другой браузер/окно и зайди под другим пользователем

### Ожидаемый результат:
- ✅ Сообщение отображается в чате
- ✅ Сообщение сохраняется в БД (SQLite через Prisma)
- ✅ Второй пользователь видит сообщение
- ✅ Статус сообщения: sent → delivered

---

## 3. ТЕСТ: 3-режимный поиск

### 3.1 Поиск по чатам пользователя
1. Нажми кнопку **"Поиск"** (MessageSquare)
2. Выбери вкладку **"По чатам"**
3. Введи название чата
4. Ожидание: показываются только мои чаты

### 3.2 Поиск по сообщениям
1. Выбери вкладку **"По сообщениям"**
2. Введи текст
3. Ожидание: показываются сообщения из моих чатов

### 3.3 Глобальный поиск
1. Выбери вкладку **"Глобальный"**
2. Выбери подрежим: **"Всё" / "Люди" / "Группы" / "Сообщества"**
3. Введи запрос (или оставь пустым для отображения всех)
4. Ожидание:
   - Если чатов < 200 - показываются все сразу
   - Поиск по людям: displayName, fullName, email
   - Поиск по группам: name, description
   - Поиск по сообществам: name, description

---

## 4. ТЕСТ: Семейные связи из контактов

### Шаги:
1. Открой **"Контакты"** (вкладка)
2. Выбери контакт
3. Нажми **"Семейные связи"** или **"Добавить связь"**
4. Выбери тип связи из выпадающего списка:
   - Друг
   - Семья
   - Коллега
   - Знакомый
   - etc.

### Ожидаемый результат:
- ✅ Семейная связь создана
- ✅ Отображается в профиле пользователя
- ✅ Зеркальная связь создаётся автоматически

## 5. ПРОВЕРКА API

### GET /api/chats?userId=xxx
```bash
curl "http://localhost:3000/api/chats?userId=user1"
```
Ожидание: список чатов пользователя

### POST /api/chats
```bash
curl -X POST http://localhost:3000/api/chats \
  -H "Content-Type: application/json" \
  -d '{"type":"private","participants":["user1","user2"],"createdBy":"user1"}'
```
Ожидание: чат создан, контакты добавлены зеркально

### GET /api/global-search?q=...&type=all&userId=...
```bash
curl "http://localhost:3000/api/global-search?q=test&type=all&userId=user1"
```
Ожидание: { users: [...], groups: [...], communities: [...] }

### GET /api/messages/search?q=...&userId=...
```bash
curl "http://localhost:3000/api/messages/search?q=test&userId=user1"
```
Ожидание: список сообщений

### GET /api/chats/search?q=...&userId=...
```bash
curl "http://localhost:3000/api/chats/search?q=test&userId=user1"
```
Ожидание: список моих чатов по поиску
