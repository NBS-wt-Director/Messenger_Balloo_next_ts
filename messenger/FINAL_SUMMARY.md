## 🔧 Независимость от Сторонних Сервисов

| Функция | Реализация | Зависимости |
|---------|------------|-------------|
| **Хранение файлов** | Яндекс.Диск API | Только OAuth токен |
| **Бэкап** | Собственное API | Нет |
| **Синхронизация** | Собственное API | Нет |
| **E2E шифрование** | Web Crypto API | Нет |
| **Демонстрация экрана** | WebRTC GetDisplayMedia | Нет |
| **Звонки** | WebRTC | Нет (только STUN) |
| **Уведомления** | Web Push API | Нет |
| **База данных** | RxDB + IndexedDB | Нет |

**Полная независимость!** Никаких Firebase, AWS, Google Cloud и т.д.

## 🚀 Запуск

```bash
cd messenger
npm install
npm run dev
```

Открыть `http://localhost:3000`

### Переменные окружения (.env.local)

```env
# Яндекс.Диск
YANDEX_DISK_OAUTH_TOKEN=ваш_токен

# Приложение
NEXT_PUBLIC_APP_URL=http://localhost:3000
NEXT_PUBLIC_VAPID_PUBLIC_KEY=ваш_VAPID_ключ
```
