## Обзор Продукта

**Balloo** - кроссплатформенный безопасный мессенджер с фокусом на приватность и удобство использования.

### Целевая Аудитория

- Частные пользователи (B2C)
- Малый бизнес (B2B)
- Организации с требованиями к безопасности

### Уникальные Особенности

- 🔒 E2E шифрование по умолчанию
- 🌐 PWA - работает в браузере как приложение
- 📱 Кроссплатформенность (Web, Mobile, Desktop)
- 🌍 Мультиязычность (4+ языка)
- 💼 Админ-панель для управления

## API Спецификация

### Auth Endpoints

```
POST   /api/auth/register       # Регистрация
POST   /api/auth/login          # Вход
GET    /api/auth/me             # Текущий пользователь
POST   /api/auth/logout         # Выход
POST   /api/auth/refresh        # Обновление токена
POST   /api/auth/yandex         # Яндекс.ID вход
```

### Chat Endpoints

```
GET    /api/chats               # Список чатов
POST   /api/chats               # Создать чат
GET    /api/chats/:id           # Данные чата
PUT    /api/chats/:id           # Обновить чат
DELETE /api/chats/:id           # Удалить чат
POST   /api/chats/:id/leave     # Выйти из чата
```

### Message Endpoints

```
GET    /api/messages            # Сообщения чата
POST   /api/messages            # Отправить сообщение
PUT    /api/messages/:id        # Редактировать
DELETE /api/messages/:id        # Удалить
PUT    /api/messages/:id/reaction # Реакция
```

### Admin Endpoints

```
GET    /api/admin/users         # Все пользователи
PUT    /api/admin/users/:id     # Обновить пользователя
DELETE /api/admin/users/:id     # Удалить пользователя
GET    /api/admin/reports       # Жалобы
PUT    /api/admin/reports/:id   # Решить жалобу
```
